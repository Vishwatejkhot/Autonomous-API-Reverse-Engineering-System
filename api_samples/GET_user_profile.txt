# Example: Get user profile

curl -X GET "https://demo-api.com/users/1/profile" \
  -H "Accept: application/json" \
  -H "Authorization: Bearer <token>"

Response:
{
  "id": 1,
  "bio": "Software engineer",
  "location": "London, UK",
  "followers": 1200,
  "following": 340
}

Error 403:
{
  "error": "UNAUTHORIZED",
  "message": "Missing or invalid token"
}
